# 🏃‍♂️ UPTOS Breza - Zvanična Web Stranica

Zvanični repozitorij za web platformu **Udruženja pedagoga tjelesnog odgoja i sporta (UPTOS)** iz Breze. 
Ova web stranica služi kao centralno informativno mjesto za roditelje, članove, sponzore i širu javnost o svim sportskim i edukativnim aktivnostima koje udruženje sprovodi.

🌐 **Live demo stranice:** [uptos.netlify.app](https://uptos.netlify.app/)

---

## 🎯 O Projektu
Cilj ove web stranice je digitalizacija i profesionalna prezentacija rada UPTOS-a. Kroz sajt, posjetioci mogu pratiti najnovije vijesti, prijaviti se za projekte i saznati više o misiji udruženja koja se temelji na promociji zdravog načina života, sporta i edukacije djece.

### Ključne funkcionalnosti (Features)
- ⛷️ **Prezentacija projekata:** Detalji o ključnim aktivnostima poput UPTOS Ski Škole, Malih olimpijskih igara i ulične utrke "Novo Kojić".
- 📰 **Sekcija sa novostima (Blog/Vesti):** Redovno ažuriranje aktuelnosti, rezultata takmičenja i dodjela priznanja.
- 📱 **Responzivni dizajn:** Stranica je u potpunosti prilagođena za pregled na mobilnim telefonima, tabletima i desktop računarima.
- 📞 **Kontakt i informacije:** Lak pristup kontakt podacima, društvenim mrežama i informacijama za donatore/sponzore.
